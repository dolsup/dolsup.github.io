Sequel Pro의 크래시 문제 대처

최신 빌드를 사용하자

2020.4.26 발행 · 2020.5.5 편집

  • Sequel Pro
  • 도구
  • macOS
  • Tip

Sequel Pro는 macOS에서 사용할 수 있는 아주 편리하고 강력한 오픈소스 MySQL, MariaDB 클라이언트 GUI 툴입니다. 저도 이 툴을 사용한지가 상당히 오래 되었습니다.

그러나 사실 Sequel Pro는 현재 몇 가지 문제를 안고 있습니다. 모하비 업데이트 이후 탭 하나를 닫으면 앱이 크래시되면서 모든 창이 종료되는 이슈가 있습니다. 이 경우 Query Editor 탭에서 작성중이던 것은 모두 사라집니다.

마지막 릴리즈는 놀랍게도 2016년 4월에 발표된 1.1.2 버전이고, 공식 릴리즈에서 이 문제는 해결될 기미가 보이지 않습니다. 다음 릴리즈 일정에 대해 묻는 이슈에서 사람들은 공식 릴리즈는 아니지만 테스트 빌드는 정상 작동하니 대안으로 이것을 사용하라고 합니다.

TL;DR

👉 Sequel Pro 테스트 빌드

실제로 최신 테스트 빌드들은 정상 작동합니다. 저도 지금은 이것을 사용중입니다. 테스트 빌드의 재미있는 점은, 공식 빌드와 달리 앱 아이콘에 ‘꿀’이 그려져있지 않다는 것입니다.

제가 직접 사용해 본 결과 공식 빌드와 현재 테스트 빌드의 눈에 띄는 기능적 변경사항은 다음과 같습니다.

  • 일단 당연히 크래시가 나지 않음
  • 테이블 조회 화면에서 검색 컴포넌트는 기본적으로 가려져 있으며 ⌘ + F를 눌러야 나옴
  • 다크 모드 지원
  • 그 외 성능 향상. 특히 Query Editor의 Syntax Highlight 기능이 상당히 최적화된 것으로 보임
