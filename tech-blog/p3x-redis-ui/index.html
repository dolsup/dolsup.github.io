개요

P3X Redis UI는 웹이나 데스크탑 앱(Electron)으로 사용할 수 있는 Redis GUI 클라이언트입니다.
GitHub Repository: https://github.com/patrikx3/redis-ui
온라인 데모: https://p3x.redis.patrikx3.com/

주의

2020년 11월 4일 기준, Sentinel을 지원하지 않습니다. 참고

웹으로 사용하기

사용할 수 있는 방법이 굉장히 다양한데 웹으로 사용하는 방법을 알아봅시다.

npm i -g p3x-redis-ui
p3x-redis

npm으로 설치할 수 있습니다. 별도의 설정 없이 실행 후 http://localhost:7843 으로 접속하면 바로 사용 가능합니다.

설정

p3x-redis-ui는 json으로 된 설정 파일 두 개를 인자로 받을 수 있습니다.

  • 설정 파일: -c 파라미터로 넘길 수 있습니다. p3xrs.json 예시
  • 커넥션 정보 파일: -n으로 넘길 수 있는 커넥션 정보 파일입니다. 그냥 실행하면 홈 디렉토리에 .p3xrs-conns.json을 생성하는데 여기에 커넥션 정보가 저장됩니다.

자세한 설정에 관해서는 redis-ui-server를 참고하세요.
